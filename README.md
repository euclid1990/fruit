# Fruit Application

## Overview
- This application built with Meteor, Angular 2
- Features
  - Authenticate
  - Realtime Update

## Setup and Run Project

```bash
$ git clone git@github.com:euclid1990/fruit.git my-proj
$ cd my-proj
$ meteor npm install
$ meteor
```

## Access Application On Local

```
http://localhost:3000/
```
